# Sonoma Animal Shelter Data Analysis
### Group Project | California State University Monterey Bay | Fall 2025
**Team:** Kanishka Harchandani, Lyn Larson, Desiree Torres
Analyzed 31,774 real-world animal shelter records from Sonoma County to answer: Do longer shelter stays reduce adoption likelihood? Performed full data cleaning, feature engineering, and visualization using Python, pandas, and seaborn. Key finding: Animals staying 3–6 months had the highest adoption rates (~65%), while animals leaving within the first week were predominantly returned to owners rather than adopted.
